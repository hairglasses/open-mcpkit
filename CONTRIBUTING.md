# Contributing

Keep this repository synthetic, portable, and reviewable.

- Do not add credentials, tokens, cookies, host-specific paths, private repo
  names, tenant data, or user data.
- Keep tools local-only and deterministic.
- Add or update tests for new command behavior.
- Run `make ci` before opening a pull request.
- Keep live send, live submit, browser automation, OAuth, and account connector
  code out of this public sample.
